Conceitos básicos sobre Vetores

Vetores são estruturas utilizadas para armazenar vários valores do mesmo tipo em uma única variável, organizados por posições chamadas índices.

O que é um Vetor?

Declaração de Vetor

Um vetor é declarado informando o tipo de dado, o nome da variável e a quantidade de posições que ele terá.

Exemplo: int numeros[5]; → Cria um vetor capaz de armazenar 5 números inteiros.

Clique aqui para saber mais

Índices do Vetor

Os vetores na linguagem C começam sempre na posição 0.

Se declararmos int v[5]; as posições válidas serão: v[0], v[1], v[2], v[3] e v[4].

Clique aqui para saber mais

Armazenamento de Dados

Todos os valores armazenados em um vetor devem ser do mesmo tipo.

Exemplo: um vetor do tipo int só pode armazenar números inteiros.
